📋 사용법 안내

개별 변환:

  1. 카카오 개발자센터에서 앱을 생성하고 REST API 키를 발급받으세요.
  2. 위의 API 설정란에 발급받은 REST API 키를 입력하세요.
  3. 변환하고 싶은 주소를 입력하고 '변환' 버튼을 클릭하세요.

일괄 변환:

  1. 엑셀 파일(.xlsx, .xls)을 준비하세요. 주소가 포함된 컬럼이 있어야 합니다.
  2. '일괄 변환' 탭에서 파일을 업로드하세요.
  3. 주소가 포함된 컬럼을 선택하세요.
  4. 변환 방향을 선택하고 '일괄 변환 시작' 버튼을 클릭하세요.
  5. 변환 완료 후 '결과 다운로드' 버튼으로 결과를 받으세요.
